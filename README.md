# Getting started
Install Dependencies:
```sh
$ bundle install
```

Create your database.yml (or just use the example):
```sh
$ cp config/database.yml.example config/database.yml
```

Create your database.yml (or just use the example):
```sh
$ rails db:migrate
```

Run the tests with RSpec
```sh
$ rake spec
```

Run the local server and see it in your browser
```sh
$ rails server
```
